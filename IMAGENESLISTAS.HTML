Lo que se ve

el atributo hret puede adoptar tres formas distintas

  • Vinculos relativos: como las imagenes, tienen una ruta usando como referencia el lugar donde se encuentra el documento html
  • Vinculo Absoluto: esto tambien es lo mismo que los imagenes, podemos usar una url completa. Siempre teniendo en cuenta no usar un link de busqueda sido la url concreta del recurso a vincular
  • Vinculo ancla: los vinculos ancla se usan para referenciar a las distintas partes del mismo documento, para utilizar se debe usar el atributo global id que debe ser unico en el documento

por otro lado los vinculos tiene otro atributo llamado target
